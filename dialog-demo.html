Custom Dialog Box System

Modern, responsive, and customizable dialog boxes to replace browser alerts

Features:

  • Modern, clean design with smooth animations
  • Responsive layout for mobile and desktop
  • Multiple dialog types (success, error, warning, info)
  • Customizable buttons and actions
  • Keyboard support (ESC to close)
  • Click outside to close (optional)
  • Promise-based API for async handling

    • Usage Examples:

    // Simple success dialog
showAlert('Operation completed successfully!', 'success');

// Error dialog
showAlert('Something went wrong. Please try again.', 'error');

// Confirmation dialog
showConfirm('Are you sure you want to delete this item?', 
    () => console.log('Confirmed'), 
    () => console.log('Cancelled')
);

// Custom dialog with multiple buttons
dialog.show({
    title: 'Custom Dialog',
    message: 'This is a custom dialog with multiple options.',
    type: 'info',
    buttons: [
        { text: 'Option 1', type: 'secondary', action: () => console.log('Option 1') },
        { text: 'Option 2', type: 'primary', action: () => console.log('Option 2') },
        { text: 'Option 3', type: 'danger', action: () => console.log('Option 3') }
    ]
});
