# Household Connect Extension Pack

This extension pack adds premium features and functionality to the Household Connect platform.

## Available Extensions

### 1. Notification System (`notifications/`)
- Real-time notifications
- Email notifications
- SMS notifications
- Push notifications for mobile

### 2. Payment Gateway Integration (`payment-gateway/`)
- Mobile Money integration (MTN, Airtel)
- Credit/Debit card processing
- PayPal integration
- Transaction management

### 3. Messaging/Chat System (`messaging/`)
- Real-time chat between users
- File sharing
- Message history
- Read receipts

### 4. Reporting & Analytics (`analytics/`)
- Dashboard statistics
- Revenue reports
- User analytics
- Export functionality

### 5. Mobile App API (`mobile-api/`)
- RESTful API endpoints
- Authentication
- Data synchronization
- Push notification support

### 6. Email Templates (`email-templates/`)
- Professional email templates
- Automated email campaigns
- Customizable templates
- Multi-language support

## Installation

1. Run the installer: `php extensions/install.php`
2. Follow the on-screen instructions
3. Configure extensions as needed

## Requirements

- PHP 8.0+
- PostgreSQL 12+
- Apache/Nginx web server
- SSL certificate (recommended)
